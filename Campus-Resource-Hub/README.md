# 🚀 CGC HUB PRO | Masterpiece Studio
### The Next-Gen Campus Resource Sharing Ecosystem for CGC University

CGC HUB PRO is an ultra-premium, high-performance desktop platform designed to transform how students at Cgc University network, share academic resources, and build a collaborative campus community. Built with a pure-Java high-end design system, it delivers a startup-grade user experience for the modern student.

---

## ✨ Outstanding Features

### 🌐 The Explore Hub
*   **Intelligent Inventory Management**: Browse books, hardware, and notes with real-time availability tracking.
*   **Academic Integration**: Filter resources by course codes (CS101, MATH202, etc.) for laser-focused studying.
*   **Contextual Direct Messaging**: Initiate instant chats with resource owners with automated context linking.

### 🎯 Skill Share & Study Groups
*   **Knowledge Marketplace**: A dedicated hub for peer-to-peer skill sharing. Find a SolidWorks coach or a tutoring partner.
*   **Collaborative Groups**: Join verified study circles directly through the platform.

### 💬 Deep Chat Infrastructure
*   **Fixed-Header Messaging**: A robust chat system with a pinned input bar and auto-scrolling message bubbles.
*   **Integrated Claim Logic**: Claim items in the **Lost & Found** section with secure proof-of-ownership attachments sent via DM.

### 📊 Professional Dashboard
*   **Campus News & alerts**: Stay updated with live news, library hours, and maker-space events.
*   **Community Reputation**: Build your campus standing with integrated "Lending Reputation" and "Verified Friends" stats.
*   **Timeline Activity**: A visual history of all your interactions and notifications.

### 🛡️ Smart Borrowing & Safety
*   **Verifiable QR Checkout**: Programmatic QR generation for secure item handovers and returns.
*   **Flexible Bookings**: Request specific durations and track approved/overdue items with ease.
*   **Local Data Resilience**: High-performance SQLite backend for rock-solid reliability.

## 🚀 Getting Started

### Prerequisites
*   **JDK 21+** (Java Development Kit)
*   Standard terminal access

### Launch Instructions (Windows)
1. Clone the repository: `git clone https://github.com/himanshuraj650/Campus-Resource-Hub.git`
2. Run the startup script:
```powershell
.\run.bat
```

### Launch Instructions (Mac/Linux)
```bash
./run.sh
```
